AIRES (AI-Based Resume Analyzer \& Job Match System) is a full-stack web application designed to streamline the recruitment process by leveraging artificial intelligence to bridge the gap between job seekers and recruiters. The platform functions as an intelligent Applicant Tracking System (ATS) that automates resume parsing, candidate ranking, and job matching using skill-based algorithms.\\\\
The system supports two primary user roles: \textbf{Candidates} and \textbf{Recruiters}. Candidates can register, upload their resumes in PDF or DOCX format, and instantly receive AI-generated insights including extracted skills, education history, and work experience. The system then automatically calculates a job match score for each available position based on skill alignment, enabling candidates to identify the most suitable opportunities at a glance. Candidates can also apply directly to jobs and track the status of their applications in real time.\\\\
Recruiters are provided with a dedicated dashboard from which they can post new job listings with associated required skills, manage existing postings, view a ranked list of all applicants sorted by their match score, and update application statuses (e.g., Shortlisted, Rejected). An analytics panel provides a high-level overview of recruitment activity, including total jobs posted, total applicants, shortlist rate, and pending reviews.\\\\
The application is built using the MERN stack---MongoDB, Express.js, React, and Node.js---with Vite as the development build tool and localStorage for client-side data persistence. The AI parsing simulation extracts structured candidate profiles from uploaded resumes, and a weighted skill-matching algorithm computes percentage-based compatibility scores between candidates and job requirements. The system architecture emphasizes modularity, scalability, and a premium user experience with responsive design and smooth transitions.\\\\
AIRES demonstrates the practical application of AI-driven tools in human resources, reducing manual screening effort while improving the quality and efficiency of the hiring pipeline.
