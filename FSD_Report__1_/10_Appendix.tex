\chapter{APPENDIX}

\section{Minimum Software Requirements}
\medskip
\textbf{Operating System}: Windows 10/11, macOS 12+, or any modern Linux distribution
\\\\
\textbf{Web Browser}: Google Chrome 110+, Mozilla Firefox 110+, or Microsoft Edge 110+
\\\\
\textbf{Node.js}: Version 18.0 or higher (for running the development server and backend)
\\\\
\textbf{npm}: Version 9.0 or higher (bundled with Node.js)
\\\\
\textbf{Code Editor}: Visual Studio Code (recommended) or any text editor with JavaScript support

\section{Minimum Hardware Requirements}
\textbf{Processor}:  Intel Core i5 (or AMD equivalent) or higher for smooth development and build operations.
\\\\
\textbf{RAM}: 8 GB (16 GB recommended for running both the frontend and backend development servers simultaneously)
\\\\
\textbf{Graphics Card}: Integrated graphics are sufficient; no dedicated GPU is required for this web application.
\\\\
\textbf{Storage}: 1 GB free disk space for the project files and \texttt{node\_modules} dependencies.
\\\\
\textbf{Display}: A minimum screen resolution of 1280x720 is recommended for the best dashboard experience. Full HD (1920x1080) or higher is ideal.
\\\\
\textbf{Input Devices}: Standard keyboard and mouse or touchpad.
\\\\
\textbf{Network Connectivity}: A stable internet connection is required for MongoDB Atlas access, cloud deployment, and any future integration with external AI/NLP APIs.
\\\\
